=======================================

核心文件:
---------
📄 index.html          - 主界面文件，包含完整的HTML结构和内联样式
📄 README.md           - 详细使用说明和安装指南

JavaScript文件:
---------------
📁 js/
  📄 main.js           - 主控制脚本，协调整个工作流程
  📄 crawler.js        - 产品信息爬虫模块，模拟数据抓取
  📄 imageGenerator.js - 图片生成模块，使用Canvas生成营销图片
  📄 fileManager.js    - 文件管理模块，处理下载和ZIP打包
  📄 demo.js           - 演示和测试脚本，提供自动化测试功能

样式文件:
---------
📁 css/
  📄 style.css         - 额外样式文件，提供动画和视觉效果

外部依赖 (CDN):
---------------

🔗 FontAwesome 6.7.2   - 图标库
🔗 Google Fonts        - 中文字体支持
🔗 ECharts 5.6.0       - 图表库（如需要）
🔗 JSZip 3.10.1        - ZIP文件生成库

功能特性:
---------
✅ 产品信息抓取（模拟）
✅ 三种类型图片生成（主图/规格图/详情图）
✅ 多种模板样式选择
✅ 真实图片下载功能
✅ ZIP批量下载
✅ 配置保存与恢复
✅ 响应式界面设计
✅ 进度显示和状态管理
✅ 演示和测试功能
✅ 性能监控工具

使用方法:
---------
1. 将所有文件保存在同一文件夹中
2. 用现代浏览器打开 index.html
3. 确保网络连接正常（需要加载CDN资源）
4. 输入淘宝产品链接开始使用

快捷键:
-------
Ctrl + Shift + D - 启动演示模式
Ctrl + Shift + T - 运行功能测试
Ctrl + Shift + P - 开始性能监控

注意事项:
---------
⚠️ 需要现代浏览器支持（Chrome 80+推荐）
⚠️ 需要网络连接加载外部CSS/JS库
⚠️ 当前版本使用模拟数据，实际应用需要后端支持
⚠️ 图片下载功能依赖浏览器下载API
⚠️ 部分功能可能需要用户手动允许下载

技术栈:
-------
🔧 HTML5 + CSS3 + JavaScript ES6+
🔧 TailwindCSS (样式框架)
🔧 Canvas API (图片生成)
🔧 LocalStorage (配置保存)
🔧 JSZip (文件打包)
🔧 Blob API (文件下载)
